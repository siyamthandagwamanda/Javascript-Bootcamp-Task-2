# Javascript-Bootcamp-Task-2

📘 JavaScript Bootcamp Task 2 – Student Result Checker

This project is a simple single-page web application built using HTML, CSS, and JavaScript.
It allows users to enter a student's name and mark, then instantly determines whether the student has passed or failed without reloading the page.

The project focuses on core DOM manipulation, form validation, and JavaScript control logic.

User enters:
    Name + Mark

          ↓
